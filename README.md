# 幸福教養｜模組1 大人先穩（PWA）

這是一個可安裝到手機桌面、可離線使用的單頁工具：
- 60 秒急救（呼吸 4/6 + 救場句）
- 年齡 × 情境切換
- 錦囊妙計（可直接急救）
- 平日微練功（每日一句）
- 每日自評（最近 7 天紀錄）

## 檔案結構
- index.html
- styles.css
- app.js
- manifest.webmanifest
- sw.js
- /icons
  - icon-192.png
  - icon-512.png

## 安裝方式
- Android：Chrome 開啟 → 右上角選單 →「安裝應用程式」或頁面出現「安裝到桌面」按鈕
- iPhone：Safari 開啟 → 分享 →「加入主畫面」

## 更新提醒
若更新後看不到新版本：
- Android Chrome：長按重新整理 或 清除網站快取
- iPhone Safari：清除該網站資料 或 用無痕模式測試
